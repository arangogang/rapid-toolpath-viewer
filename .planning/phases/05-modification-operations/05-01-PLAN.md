    - OffsetPointsCommand: redo() adds delta np.ndarray to pos of each indexed EditPoint; undo() restores saved old positions. Emits model.points_changed.
    - SetPropertyCommand: accepts field name ("speed"|"zone"|"laser_on"), list of indices, new value. redo() sets new value on all; undo() restores old values. Emits model.points_changed.
    - DeletePointsCommand: accepts list of indices and mode ("reconnect"|"break"). redo() sets deleted=True on each point; for "break" mode also sets laser_on=False on the first non-deleted point after each deleted span. undo() restores deleted=False and original laser_on values. Emits model.points_changed.
    - InsertPointCommand: accepts model, source_index, delta np.ndarray. redo() creates new EditPoint with pos=source.pos+delta, copies speed/zone/laser_on/move_type from source, inserts at source_index+1 in model._points. undo() removes the inserted point. Emits model.points_changed.
    - All commands import from PyQt6.QtGui import QUndoCommand (not QtWidgets).
  </behavior>
  <action>
    Create `src/rapid_viewer/ui/commands.py` with `from __future__ import annotations` and these four classes:

    1. **OffsetPointsCommand(QUndoCommand)**:
       - `__init__(self, model: EditModel, indices: list[int], delta: np.ndarray, parent: QUndoCommand | None = None)`
       - Constructor saves `self._old_positions = [model.point_at(i).pos.copy() for i in indices]` and `self._delta = delta.copy()`
       - `redo()`: for each index, `model.point_at(idx).pos += self._delta`; then `model.points_changed.emit()`
       - `undo()`: for each index, `model.point_at(idx).pos[:] = self._old_positions[i]`; then `model.points_changed.emit()`
       - Description text: `f"Offset {len(indices)} point(s)"`

    2. **SetPropertyCommand(QUndoCommand)**:
       - `__init__(self, model: EditModel, indices: list[int], field: str, value: str | bool)`
       - Constructor saves `self._old_values = [getattr(model.point_at(i), field) for i in indices]`
       - `redo()`: `setattr(model.point_at(idx), field, self._value)` for each index; emit `points_changed`
       - `undo()`: `setattr(model.point_at(idx), field, self._old_values[i])` for each; emit `points_changed`
       - Description: `f"Set {field} ({len(indices)} point(s))"`

    3. **DeletePointsCommand(QUndoCommand)**:
       - `__init__(self, model: EditModel, indices: list[int], mode: str)` where mode is "reconnect" or "break"
       - Constructor saves: `self._old_deleted = [model.point_at(i).deleted for i in indices]`
       - For "break" mode, also identify and save the `laser_on` state of the first non-deleted point after each deleted span (scan forward from max(indices)+1 to find it)
       - `redo()`: set `deleted=True` on all indices. For "break": set `laser_on=False` on the identified next-point. Emit `points_changed`.
       - `undo()`: restore `deleted` flags from `_old_deleted`. For "break": restore saved `laser_on` values. Emit `points_changed`.
       - Description: `f"Delete {len(indices)} point(s) ({mode})"`

    4. **InsertPointCommand(QUndoCommand)**:
       - `__init__(self, model: EditModel, source_index: int, delta: np.ndarray)`
       - Constructor computes: `source = model.point_at(source_index)`, creates `self._new_point = EditPoint(original=source.original, pos=source.pos.copy() + delta, speed=source.speed, zone=source.zone, laser_on=source.laser_on)`, stores `self._insert_index = source_index + 1`
       - `redo()`: `model._points.insert(self._insert_index, self._new_point)`; emit `points_changed`
       - `undo()`: `model._points.pop(self._insert_index)`; emit `points_changed`
       - Description: `"Insert point"`

       ```python
       def apply_offset(self, indices: list[int], delta: np.ndarray) -> None:
           from rapid_viewer.ui.commands import OffsetPointsCommand
           cmd = OffsetPointsCommand(self, indices, delta)
           self._undo_stack.push(cmd)

       def set_property(self, indices: list[int], field: str, value: str | bool) -> None:
           from rapid_viewer.ui.commands import SetPropertyCommand
           cmd = SetPropertyCommand(self, indices, field, value)
           self._undo_stack.push(cmd)

       def delete_points(self, indices: list[int], mode: str) -> None:
           from rapid_viewer.ui.commands import DeletePointsCommand
           cmd = DeletePointsCommand(self, indices, mode)
           self._undo_stack.push(cmd)

       def insert_after(self, source_index: int, delta: np.ndarray) -> int:
           from rapid_viewer.ui.commands import InsertPointCommand
           cmd = InsertPointCommand(self, source_index, delta)
           self._undo_stack.push(cmd)
           return source_index + 1  # index of newly inserted point
       ```

    3. **Add `build_edited_moves()` method** that synthesizes a list of `MoveInstruction` from current EditPoint state, skipping deleted points, reflecting modified pos/speed/zone/laser_on:

       ```python
       def build_edited_moves(self) -> list[MoveInstruction]:
           from rapid_viewer.parser.tokens import MoveInstruction, RobTarget
           result = []
           for point in self._points:
               if point.deleted:
                   continue
               if point.original.target is None:
                   result.append(point.original)
                   continue
               new_target = RobTarget(
                   name=point.original.target.name,
                   pos=point.pos.copy(),
                   orient=point.original.target.orient,
                   confdata=point.original.target.confdata,
                   extjoint=point.original.target.extjoint,
                   source_line=point.original.target.source_line,
               )
               new_move = MoveInstruction(
                   move_type=point.original.move_type,
                   target=new_target,
                   circle_point=point.original.circle_point,
                   joint_target=point.original.joint_target,
                   speed=point.speed,
                   zone=point.zone,
                   tool=point.original.tool,
                   wobj=point.original.wobj,
                   source_line=point.original.source_line,
                   has_cartesian=point.original.has_cartesian,
                   laser_on=point.laser_on,
               )
               result.append(new_move)
           return result
       ```
