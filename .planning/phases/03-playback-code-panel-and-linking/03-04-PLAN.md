       # Wire signals
       self._wire_signals()

3. **_wire_signals() method:**
   ```python
   def _wire_signals(self) -> None:
       # 3D click -> select waypoint -> update code panel
       self._gl_widget.waypoint_clicked.connect(self._playback_state.set_index)

       # Current waypoint changed -> update 3D highlight + code panel
       self._playback_state.current_changed.connect(self._on_waypoint_changed)

       # Code panel click -> find matching move -> select in 3D
       self._code_panel.line_clicked.connect(self._on_code_line_clicked)
   ```

4. **_on_waypoint_changed(index: int) slot:**
   - `self._gl_widget.set_highlight_index(index)`
   - Get current move: `move = self._playback_state.current_move`
   - If move: `self._code_panel.highlight_line(move.source_line)`

5. **_on_code_line_clicked(line: int) slot:**
   - Search `self._playback_state._moves` for a move whose `source_line == line`
   - If found at index i: `self._playback_state.set_index(i)`
   - This completes LINK-02 (code click -> 3D select)

6. **_on_proc_changed(proc_name: str) slot:**
   - If proc_name == "All PROCs": use all moves from parse_result
   - Else: filter moves by checking `move.source_line` within `self._parse_result.proc_ranges[proc_name]`
   - Call `self._playback_state.set_moves(filtered_moves)`
   - Rebuild GL geometry: `self._gl_widget.update_scene_filtered(filtered_moves)` OR simpler: create a filtered ParseResult copy and call `self._gl_widget.update_scene(filtered_result)`
   - Simpler approach: store filtered moves in PlaybackState, and rebuild geometry. Create a helper method `_apply_proc_filter(proc_name)` that:
     a. Gets moves from self._parse_result.moves
     b. If proc_name != "All PROCs", filters by proc_ranges
     c. Calls self._playback_state.set_moves(filtered)
     d. Creates a filtered ParseResult (copy with filtered moves) and calls self._gl_widget.update_scene()

7. **Update load_file():**

     - `self._code_panel.set_source(self._parse_result.source_text)`

     - `self._playback_state.set_moves(self._parse_result.moves)`
