   ```python
   class CodePanel(QWidget):
       """Read-only RAPID code viewer with syntax highlighting and line highlight.

       Public API:
         set_source(text: str) -- load RAPID source text
         highlight_line(line_number: int) -- highlight and scroll to 1-indexed line
         get_cursor_line() -> int -- return 1-indexed line at current cursor position

       Signals:
         line_clicked(int) -- emitted when user clicks a line (1-indexed line number)
       """
   ```
   - Contains a `QPlainTextEdit` (self._editor) in a QVBoxLayout
   - `set_source(text)`: calls `self._editor.setPlainText(text)`, attaches `RapidHighlighter(self._editor.document())` if not already attached. Store highlighter in `self._highlighter` to prevent GC.
   - `highlight_line(line_number: int)`: 1-indexed. Use `editor.document().findBlockByLineNumber(line_number - 1)` to get block. Create `QPlainTextEdit.ExtraSelection()`, set background to `QColor("#264F78")`, set `QTextCharFormat.Property.FullWidthSelection` to True per Pitfall 3. Set cursor, call `setExtraSelections([selection])`, then `setTextCursor(cursor)` and `centerCursor()` to scroll.

   - Signal `line_clicked = pyqtSignal(int)`: connect `self._editor.cursorPositionChanged` to an internal slot that emits `line_clicked(get_cursor_line())`
   - Set editor read-only: `self._editor.setReadOnly(True)`
   - Set monospace font: `QFont("Consolas", 10)` (Windows) with fallback to "Courier New"
   - Set tab stop width to 4 spaces equivalent
