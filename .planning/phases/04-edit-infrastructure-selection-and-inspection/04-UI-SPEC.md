### OpenGL 3D Viewport Colors (RGB float 0.0-1.0)

| Element | Color | RGB Float | Usage |
|---------|-------|-----------|-------|
| Unselected marker | Yellow | `(1.0, 1.0, 0.3)` | Default waypoint markers (existing COLOR_MARKER) |
| Selected marker | Cyan | `(0.0, 1.0, 1.0)` | Waypoints in SelectionState.selected set |
| Current marker | White | `(1.0, 1.0, 1.0)` | PlaybackState.current_index highlight (existing) |
| Current+selected marker | Magenta | `(1.0, 0.3, 1.0)` | current_index when also in selection set |
| Path lines | Unchanged | See geometry_builder.py | MoveL=green, MoveJ=orange, MoveC=blue |

Selection marker point size: 10.0 (same as regular markers).
Current marker point size: 14.0 (existing highlight, larger to distinguish).

Source: D-05 from 04-CONTEXT.md -- "Selection visual feedback uses color changes only, no size or shape changes."

## Interaction Contract

### Single Click (no modifier)

1. Ray-cast pick waypoint (existing 20px threshold)
2. `PlaybackState.set_index(clicked_index)` -- moves playback cursor
3. `SelectionState.set_selection({clicked_index})` -- replaces selection with single point
4. GL widget renders clicked marker as white (current), all others yellow
5. Code panel scrolls to corresponding source line (existing behavior)
6. Property panel updates to show clicked point's properties

Source: D-02 from 04-CONTEXT.md

### Shift+Click

1. Ray-cast pick waypoint
2. `PlaybackState.set_index(clicked_index)` -- moves playback cursor
3. `SelectionState.toggle(clicked_index)` -- adds or removes from selection set
4. GL widget renders: selected markers cyan, current marker white or magenta (if also selected)
5. Code panel scrolls to clicked point's source line
6. Property panel updates to show clicked point's properties, header shows count

### Ctrl+Click

Same behavior as Shift+Click (both modifiers toggle selection).

Source: D-02 from 04-CONTEXT.md

### Click on Empty Space

1. Ray-cast finds no waypoint within threshold
2. Selection unchanged (do NOT clear selection on empty click)
3. PlaybackState unchanged

### Keyboard Shortcuts

| Shortcut | Action | State |
|----------|--------|-------|
| Ctrl+Z | Undo | Disabled in Phase 4 (no edit commands yet) |
| Ctrl+Y | Redo | Disabled in Phase 4 (no edit commands yet) |
| Ctrl+O | Open file | Existing |
| Ctrl+Q | Exit | Existing |
